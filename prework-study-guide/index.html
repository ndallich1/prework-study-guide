HTML

  • The head element contains information about the webpage.
  • The body element represents the visible context shown to the user.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The link element conneccts an external resource to the HYML document
  • The base element defines the base URL for the website
  • The style element defines the CSS styles associated with a website
  • the title of a webpagecan be seen in the page's tab or browser title bar
  • Child elements are elements nested inside other elements
  • Parent element is an element that holds all child elements
  • Charset: sets our character encoding
  • http-equiv: sets an HTTP header for content
  • viewport: sets the default view for our webpage
  • Tags: what is inside angle brackets (<>)
  • h1, h2, h3, etc.: heading levels. h1 should only appear on a page once!
  • p: paragraph
  • ul: unordered list
  • ol: ordered list
  • li: list items (like a bullet point)
  • src: defines location of an image
  • br: line break
  • a: creates a link to the same webpage or other webpages (anchor)
  • href: points to the URL for the link
  • alt: contains text string that describes the appearance and funtionality of the image
  • section elements: semantic elements that define a section in a document that contains thematic content.
