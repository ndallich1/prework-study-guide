HTML

  • Hypertext Markup Language (HTML) is the basic building block of a website
  • HTML is one of the (3) core languages of the web
  • HTML handles the context of a website and the structure or meaning of each element on a page
  • The head element contains information about the webpage.
  • The body element represents the visible context shown to the user.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The link element conneccts an external resource to the HYML document
  • The base element defines the base URL for the website
  • The style element defines the CSS styles associated with a website
  • the title of a webpagecan be seen in the page's tab or browser title bar
  • Child elements are elements nested inside other elements
  • Parent element is an element that holds all child elements
  • Charset: sets our character encoding
  • http-equiv: sets an HTTP header for content
  • viewport: sets the default view for our webpage
  • Tags: what is inside angle brackets (<>)
  • h1, h2, h3, etc.: heading levels. h1 should only appear on a page once!
  • p: paragraph
  • ul: unordered list
  • ol: ordered list
  • li: list items (like a bullet point)
  • src: defines location of an image
  • br: line break
  • a: creates a link to the same webpage or other webpages (anchor)
  • href: points to the URL for the link
  • alt: contains text string that describes the appearance and funtionality of the image
  • section elements: semantic elements that define a section in a document that contains thematic content.

CSS

  • CSS defines the colors, aesthetics, visual layout, and organization of a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS: inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. This is good for quick changes, but less flexible than internal and external style sheets.
  • Internal Style Sheet: embed an entire style sheet directly in the HTML using a style element in the head element.
  • External Style Sheet: all CSS rules are kepy in a separate file. This makes design changes easier. External Style Sheets are most common.
  • Declarations: contain the CSS property we want to apply and the value of the property
  • Selector: an element that CSS will apply
  • (DRY) code: DO NOT REPEAT YOURSELF
  • rel: specifies the relationship between the current document and the linked resource document
  • href: specifies the location (URL) of the external resource
  • Class Attributes: allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
